# Loomé Fashion Website

A modern luxury fashion landing page built using pure HTML and CSS with an editorial-style UI inspired by premium ecommerce websites.

---

## Live Demo

🔗 [Click Here To Visit The Website](https://CodeByDiablo.github.io/fashion-website/)

---

## Features

- Modern luxury UI
- Editorial-style layout
- Flexbox-based structure
- Floating circular action button
- Rotated typography effect
- Premium whitespace usage
- Product showcase layout
- Vertical thumbnail gallery
- Feature cards section
- Minimal luxury color palette
- Clean responsive structure

---

## Technologies Used

- HTML5
- CSS3
- Remix Icons

---

## Project Structure

```bash
Loomé-Fashion-Website/
│
├── index.html
└── style.css
```

---

## Design Approach

This project follows a modern editorial-style frontend structure using:

- Flexbox layouts
- Absolute positioning
- Viewport typography (`vw`)
- Floating UI elements
- Minimal luxury aesthetics
- Soft neutral color palette
- Editorial composition techniques

---

## How To Run Locally

1. Download or clone the repository

```bash
git clone https://github.com/CodeByDiablo/fashion-website.git
```

2. Open the project folder

3. Run `index.html`

OR

Use VS Code Live Server.
